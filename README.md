#### if条件中对整数的判断 if(a)
遵从的原则是非0即为真
	
	int a1 = 2, a2 = 0, a3 = -2;
	if(a1) // 返回true
	if(a2) // 返回false
	if(a3) // 返回true

#### 使用位运算代替整数的乘法（左移）和除法（右移）
如果需要乘以或除以2n，都可以用移位的方法代替

	int a1 = 3;
	int a2 = a1 * 2; // a2 = a1 << 1;
	int a3 = a1 / 2; // a2 = a1 >> 1;

#### 使用位运算代替整数的取模（与运算）
如果需要和2n取模，都可以用与运算的方法代替

	int a1 = 3;
	int a2 = a1 % 4; // a2 = a1 & 3;

#### 避免不必要的除法
* 加法和减法，速度快于乘法和除法
* 避免连续除法（改用倒数的乘法）
